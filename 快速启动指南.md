# 快速启动指南

## 第一步：准备文件

### 1. 移动PDF简历文件
请将项目根目录下的 `林森-机器人开发-简历.pdf` 移动到 `public/` 目录中：

```bash
# Windows PowerShell
Move-Item "林森-机器人开发-简历.pdf" "public/"

# 或手动复制到 public 目录
```

### 2. 添加图片和视频资源
请根据 `README.md` 中的详细说明，将所需的图片和视频文件添加到以下目录：
- `public/images/` - 所有图片资源
- `public/videos/` - 视频资源

## 第二步：安装依赖

```bash
npm install
```

## 第三步：启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 http://localhost:3000

## 第四步：构建生产版本

```bash
npm run build
```

构建完成后，生产文件在 `dist/` 目录中。

## 注意事项

- 所有图片建议使用 WebP 格式以优化加载速度
- 视频文件大小应控制在 50MB 以内
- 确保图片和视频文件名与代码中的路径一致
- 如果图片缺失，网站会自动显示占位符

## 常见问题

**Q: 图片显示不出来？**  
A: 请检查图片路径是否正确，图片是否已放入 `public/images/` 目录。

**Q: 视频无法播放？**  
A: 请确保视频文件是 MP4 格式，并且已放入 `public/videos/` 目录。

**Q: PDF简历下载失败？**  
A: 请确保PDF文件已移动到 `public/` 目录，文件名与代码中的路径一致。
